# AI Mini Clinic Chatbot

## Overview
AI-powered healthcare chatbot built using Python and Natural Language Processing (NLP).  
The chatbot guides users through structured symptom-based workflows and provides preliminary health suggestions.

This project demonstrates:
- AI workflow design
- Prompt structuring
- Conversational logic
- NLP preprocessing
- User interaction flow optimisation

## Features
- Conversational symptom checker
- Intent recognition
- Structured prompt workflow
- Clean modular Python design
